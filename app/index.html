Digital dialog fastlege | Helse- og sosialtjenester

Digital dialog fastlege er en utvidelse av webportalen "Min helse". I digital dialog fastlege vil man kunne kommunisere med sin fastlege via meldinger, bestilling av timer, fornyelse av respeter og e-konsultasjon. Det er integrasjoner mot andre system som fastlegen bruker. Webportalen er basert på moderne teknologi som HTML5 og ReactJS som kommuniserer mot REST service i WCF og backend i .NET. NPM ble brukt som pakkeverktøy og Gulp byggverktøy frontend. Det er strenge krav til sikkerhet og universell utforming.
